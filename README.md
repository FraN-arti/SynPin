## Как это работает

SynPin — это платформа, где ИИ-агенты живут в организации как настоящие сотрудники. У каждого агента есть имя, роль и место в команде. Они общаются друг с другом, делятся задачами и помнят прошлые разговоры.

В отличие от обычных чат-ботов, здесь агенты **действительно работают вместе**: один пишет код, другой проверяет, третий помогает с архитектурой. И всё это — в одном интерфейсе.

## Возможности

### Живой чат с инструментами

Агенты отвечают в реальном времени и выполняют действия: работают с файлами, запускают код, ищут информацию. Каждое действие видно в интерфейсе.

### Команда с характером

Каждый агент — не безликий бот, а сотрудник со своей специализацией и манерой общения. Один — строгий аналитик, другой — дружелюбный помощник.

### Делегирование задач

Управляющий может поручить задачу исполнителю, дождаться результата и принять решение. Настоящая иерархическая коммуникация.

### Память разговоров

Агенты запоминают прошлые обсуждения. Даже после перезапуска они помнят, что было решено и какие ошибки допускали.

### Гибкая настройка

Подключайте разные ИИ-модели для разных агентов. Настраивайте роли, департаменты и инструменты через удобный интерфейс.

## Быстрый старт

```powershell
git clone https://github.com/FraN-arti/SynPin.git
cd SynPin
pip install -e ./core
synpin start
```

Откройте `http://localhost:2088` — ваша команда готова к работе.
