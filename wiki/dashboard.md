# 📊 Dashboard

## Стек

| Компонент | Технология |
|-----------|-----------|
| Фреймворк | React 19 + Vite 6 |
| Язык | TypeScript 5.7 (strict) |
| Стили | Tailwind CSS 4 |
| State (клиент) | Zustand |
| State (сервер) | TanStack Query |
| Real-time | WebSocket |
| HTTP | Axios |

## Страницы

### 1. Главная — Overview
- Статус всех агентов в реальном времени
- Активные задачи и прогресс
- Быстрые действия (новая задача, настройка)

### 2. Агенты
- Список агентов с ролями и статусами
- Память каждого агента (ошибки, решения)
- Включение/выключение агентов
- Настройка промптов и инструментов

### 3. Задачи
- Список задач со статусами
- Детали: кто делает, что обсуждают, результат
- История выполненных задач
- Создание новых задач

### 4. Group Chat (обсуждения)
- Real-time поток сообщений между агентами
- Визуализация: кто → кому, мысль → действие
- Принятые решения и консенсусы
- Фильтрация по задаче/агенту

### 5. Memory (знания)
- Коллективная память команды
- Поиск по знаниям (семантический)
- Журнал ошибок и решений
- Экспорт/импорт

### 6. Settings
- Выбор LLM-модели и провайдера
- Настройка режимов Router (делегат/команда/группа)
- Конфигурация ролей агентов
- MCP-серверы и инструменты

## Виджеты статусов

| Статус | Цвет | Значение |
|--------|------|----------|
| 🟢 Active | Green | Агент работает |
| 🟡 Thinking | Yellow | Думает / ждёт LLM |
| 🔴 Error | Red | Ошибка |
| ⚪ Idle | Gray | Ожидает задачу |
| 💬 Discussing | Blue | Участвует в обсуждении |

## Real-time

WebSocket подключается к `/api/ws/events` и получает:
- `agent_status` — смена статуса агента
- `message` — сообщение в групповом чате
- `task_update` — обновление задачи
- `error` — ошибка агента
- `memory_update` — новая запись в памяти
