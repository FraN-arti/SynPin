## 🔗 Связи между документами

```
Configuration
    ↓
Agents (личность + скиллы + роли)
    ↓
Channels (где работают агенты)
    ↓
Kanban (что делают агенты)
    ↓
Memory (что запоминают)
```

Каждый документ ссылается на связанные. Начни с Configuration → Agents → Channels.
