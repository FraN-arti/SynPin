# 🧠 Memory System

## Проблема

В существующих фреймворках:
- Память опциональна
- Каждый запуск с нуля
- Ошибки забываются
- Нет коллективного обучения
- Ключевой поиск вместо семантического

## Наше решение

**ChromaDB** — векторная база данных для семантического поиска по памяти.

### Почему не SQLite?

| | SQLite | ChromaDB |
|--|--------|----------|
| Поиск | Ключевой/текстовый | **Семантический** (embeddings) |
| Вопрос | "Записи с auth" | "Как защитить API?" → находит про auth |
| Similarity | ❌ | ✅ Косинусное расстояние |
| Масштаб | Хорошо | Лучше для векторов |

SQLite можно использовать как дополнение для метаданных, но основной поиск — через ChromaDB.

## Слои памяти

### 1. Per-Agent Memory
Каждый агент хранит:
- **Ошибки** — что пошло wrong и какой урок извлечён
- **Решения** — успешные подходы к проблемам
- **Контекст задач** — история работы
- **Предпочтения пользователя** — стиль, формат, требования

### 2. Shared Memory (коллективная)
- Командные знания — доступны всем агентам
- Best practices — проверенные решения
- Паттерны — архитектурные решения
- Ошибки команды — чтобы не повторять

### 3. Error Log (журнал ошибок)
- Каждая ошибка с контекстом и решением
- Теги: тип, агент, задача, серьёзность
- Верификация: решение проверено / не проверено

## Как работает

### Запись ошибки

```python
memory.log_error(
    agent="programmer",
    error="Забыл auth middleware в API endpoint",
    context="Создавал POST /api/tasks",
    solution="Всегда добавлять auth middleware первым",
    tags=["security", "api", "middleware"]
)
```

### Семантический поиск

```python
# Агент начинает задачу → ищет в памяти
results = memory.search(
    query="Как защитить API endpoint?",
    n_results=5,
    threshold=0.7,        # баланс релевантности
    source="shared"       # per-agent | shared | all
)

# → Найдёт:
# 1. "JWT + refresh tokens для API auth" (85%)
# 2. "Всегда добавлять auth middleware первым" (78%)
# 3. "Rate limiting для защиты от brute force" (65%)
```

### Коллективное обучение

```python
# Ошибка одного → знание всех
error = memory.get_error(error_id=123)
memory.share_to_all(error, agents=["architect", "reviewer", "tester"])

# Теперь все агенты знают про эту ошибку
```

## Метаданные

Каждая запись в ChromaDB хранит:

```json
{
  "id": "mem_001",
  "content": "Всегда добавлять auth middleware первым",
  "type": "solution",
  "agent": "programmer",
  "tags": ["security", "api"],
  "verified": true,
  "created_at": "2026-05-31T18:00:00Z",
  "task_id": "task_042"
}
```

ChromaDB embedding (768-dim от `nomic-embed-text-v1.5`) + metadata для фильтрации.
