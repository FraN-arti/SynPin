# 📅 Roadmap

## Фаза 0: Фундамент (сейчас)

- [x] Имя: **SynPin**
- [x] Структура проекта: `core/` + `web/`
- [x] Python core: FastAPI, uvicorn, pydantic, chromadb
- [x] React UI: React 19, Vite 6, TypeScript 5.7, Tailwind 4
- [x] Dev-супервизор: `core/dev_server.py` — единый запуск
- [x] CLI: `synpin start/stop/status/setup/logs/version/update`
- [x] Install script: `scripts/install.ps1` — установка + авто-update
- [x] Конфигурация: `~/.synpin/config/` (settings, providers, memory, agents, tools)
- [x] Wiki: философия, архитектура, конфигурация

## Фаза 1: Ядро (MVP)

### Память (приоритет №1)
- [x] Базовая память: MEMORY.md + USER.md на агента (как у Hermes)
- [x] Frozen Snapshot: системный промпт фиксируется при старте
- [x] Датированные факты: facts/YYYY-MM-DD_topic.md
- [x] Shared MEMORY.md: коллективное обучение
- [x] File Locking: безопасный конкурентный доступ
- [x] Threat Scanning: защита от injection
- [x] Сессии: персистентная история чатов (JSON на диске)
- [x] state.json: bookmarks per agent (восстановление после рестарта)
- [x] Компакция: truncate при превышении context_window (настраивается)
- [x] Авто-сброс сессий (daily/timer/time + архивация)
- [x] USER.md глобальный: общая для всех агентов (shared/USER.md)
- [x] Авто-запись памяти: агент proactively сохраняет через memory_write
- [ ] **Аккаунты пользователей**: персональный USER.md per user (替代 глобального)

### Агенты
- [ ] Базовый класс Agent (роль, промпт, инструменты)
- [ ] ReAct-луп (think → act → observe)
- [ ] LLM-провайдеры (LM Studio, Ollama, OpenAI)
- [ ] Первые роли: Архитектор, Программист

### Router
- [ ] Конфигурация режимов (делегат / команда / группа)
- [ ] Каналы отделов с главами (moderation, @mention)
- [ ] Совет отделов (cross-channel делегирование)
- [ ] Совет директоров (стратегия)
- [ ] Простой делегат: задача → агент → результат
- [ ] Командный режим: обсуждение → консенсус → действие
